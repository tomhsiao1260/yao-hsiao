# 桌遊

年底跟朋友玩桌遊，好久沒玩了，感覺格外新鮮，於是想寫篇心得。另外，這裡不會有太多規則描述，更多的是遊玩當下的主觀感受。那我們就開始吧，Link Start！

我們玩了三款桌遊：Project L、德國蟑螂、阿瓦隆。對我來說，每個遊戲的開始，就像是進入作者精心設計的世界。Project L 是個蒐集越多方塊就越開心的世界，那裡不需要太多的交流，你可以照自己的步調推敲，並從中獲得成就感；德國蟑螂是個充滿欺騙的世界，那裡不需要太多的理性分析，更重要的是語氣、直覺和話語間的微妙停頓；阿瓦隆是個結合上述兩者的世界，理性的推敲和感性的判斷，讓一切變得複雜且有趣了起來。

對我來說，阿瓦隆有趣的點在於，陣營間的對抗以及角色間的資訊不對等。遊戲一開始，梅林或邪惡陣營握有的資訊相對充裕，反觀亞瑟的忠誠就得從頭摸索起。邪惡陣營要想辦法製造混亂，如果讓正義陣營達成共識，明朗的局勢會讓輿論難以動搖。亞瑟的忠誠得在混亂的輿論下，從頭建立自己的認知。儘管梅林知道很多，但如果不能以亞瑟忠誠的視角展開對話，會造成很多的認知落差。你可以很理性的分析每個環節，直到發現某個玩家的某一步並不是那麼理性，使得你得到了完全錯誤的結果；你也可以很感性的相信自己的每個判斷，直到真相揭曉才發現自己的無知。正是這些元素的交織抗衡，讓遊戲變得更加的有趣。

這讓我開始好奇，什麼樣的規則設計能展開有趣的遊玩體驗？許多不起眼的規則，對整個遊戲的風格和體驗有著決定性的影響。好比說，在玩德國蟑螂時，我們誤讓猜對的玩家在下個回合出牌，這項改動讓牌少的玩家陷入兩難，使原本純粹的吹牛遊戲融入了微妙的鬥智風格。此外，如果在 Project L 之上，多加一條奪取其他玩家方塊的規則，也不難想像遊戲風格會發生很大的轉變。

在體驗上，其實不難發現桌遊有個普遍的短板，就是最初的規則解釋需要很多的時間。這很大程度是為了確保在遊玩時，大家都遵守這套遊戲所規定的秩序。這有時候很繁瑣，因為大部分的人在遊玩過一遍後，都能對規則有一定的掌握。對我來說，最理想的遊玩體驗應該要像物理定律那樣，你不需要一直詢問自己是不是違反了物理定律，而且你可以透過直接體驗去了解它，不見得要事先讀過一遍規則細節。

最後，終於在跨年的前一小時把文章寫完，大家明年見！
