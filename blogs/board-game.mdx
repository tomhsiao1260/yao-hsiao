# 桌遊

年底跟朋友玩桌遊，因為好久沒玩了，感覺格外新鮮，於是想寫篇心得記錄。另外，相較於規則的介紹，我會花更多篇幅描述遊玩的主觀體驗。那就開始吧，Link Start！

## 桌遊世界

對我來說，每個遊戲的開始，就像是進入作者精心設計的世界。我們玩了三款桌遊：Project L、德國蟑螂、阿瓦隆。其中，Project L 就是個蒐集越多方塊越開心的世界，不需要太多的交流，可以照自己的步調推敲；德國蟑螂是個無處不在吹牛的世界，那裡不需要太多的理性分析，重要的是語氣、直覺和話語間的微妙停頓；阿瓦隆則是個結合上述兩者的世界，理性的推敲和感性的判斷，發展也變得複雜且有趣了起來。

## 阿瓦隆

對我來說阿瓦隆有趣的點在於，陣營間的對抗和角色間的資訊不對等。遊戲一開始，梅林或邪惡陣營握有的資訊相對充裕，反觀亞瑟的忠誠就得從頭摸索。邪惡陣營要想辦法製造混亂，如果讓正義陣營達成共識，局勢明朗會讓輿論難以動搖。所以亞瑟的忠誠得在混亂的輿論下，從頭建立自己的認知，這有時候並不是那麼容易。此外，儘管梅林知道很多，但如果不能以亞瑟忠誠的視角展開對話，會造成很多認知落差。你可以很理性的分析每個環節，直到發現某個玩家的某一步並不是那麼的理性，使得你得到了完全錯誤的結果；也可以很感性的相信自己的每個判斷，直到真相揭曉才發現自己的無知。正是這些元素的抗衡交織，讓遊戲變得更加的有趣。

## 規則的設計

這讓我開始好奇，什麼樣的設計能展開有趣的遊玩體驗？許多不起眼的規則，對整個遊戲風格有著決定性的影響。像是在玩德國蟑螂時，我們誤讓猜對的玩家在下個回合出牌，這個改動讓牌少的玩家陷入兩難，使原本的吹牛遊戲融入了微妙的鬥智元素。如果在 Project L 之上，多加一條奪取玩家方塊的規則，也不難想像遊玩風格會發生很大的轉變。

在體驗上，其實不難發現桌遊有個普遍的短板，就是規則的解釋需要很多時間，這很大程度是為了確保大家能遵守遊戲所規定的秩序。這有時候有點繁瑣，因為大部分的人在玩過一遍後，都能對規則有一定的掌握。對我來說，最理想的遊玩體驗應該要像物理定律那樣。你不需要一直詢問自己是不是違反了物理定律，因為在預設上一定是遵循的。而且你能透過直接的體驗去了解它，不見得要先讀過一遍規則的細節。
