# 一些小時候用電腦的記憶

這篇是生活文，主要是一些小時候讓我印象深刻，跟電腦有關的記憶。

還記得那時候桌上型電腦剛出來，連智慧型手機都沒有，我幾乎每天下課回家都待在唯一有電腦的那個房間，不為別的，就為了玩遊戲，那是段快樂的時光。

但那是我爸的電腦，只能在他下班前偷偷玩，後來我媽擔心我沉迷，設了密碼，我爸還在電腦旁放了本厚厚的科普書，他說只要能讀完，並答對他抽考的問題，就能告訴我密碼，那是段難熬的時光。

但我還是很想玩遊戲，這促使我去翻那本厚重的書，裡面有光纖、電磁波、火箭、網際網路的解釋，密密麻麻的文字讓我卻步。有次我爸說能開書考，他問的問題我至今都還記得：「網址前三個 w 分別是哪三個字的縮寫？」我沒能答出來，但 World Wide Web 這三個字也因此從小記到了現在。

但我還是很想玩遊戲，這促使我開始想破解密碼，嘗試不同的排列組合。甚至是拆掉不同的主機線，了解背後的運作原理。因為什麼事都做不了，所以總會好奇有哪些還沒試過的組合。在那漫長的時光裡，我只成功破解過一次密碼，那是在房間角落的一串不起眼的數字。當時的我真的雀躍極了，玩了整個下午的遊戲，但到了隔天，密碼被改了，就再也沒登進去過了。

還記得當時學校的電腦課，是全班一起偷玩電腦的快樂時光，老師教的也忘得差不多了，但有個小插曲記到了現在。那時候老師在解釋瀏覽器的搜尋欄和網址輸入欄的差別，她說網址欄只能輸網址，不能輸入想搜尋的問題，接著她演示了一遍並愣了一下，原來更新後的瀏覽器已經支援了這項搜尋功能。

小時候在公園玩也是段快樂的時光，記得有次假日在公園認識一個差不多年紀的小孩，他邀我去他家玩電腦。因為我媽擔心我亂跑，平常不太走公園以外的路，久了就成了習慣。所以對我來說，那天真的是開了新眼界，沿路看到好多不同的街景巷弄。不只如此，除了玩整個下午的遊戲外，他爸爸還會跟我們討論遊戲的內容。就這樣一路玩到了傍晚，擔心太晚，於是跑著回家，那是個充滿驚奇的一天！

這大概就是小時候讓我印象深刻，跟電腦有關的記憶了吧，你呢？
