# 資料標註心得

最近在幫 Vesuvius Challenge 做資料的標註，目標是將肉眼能辨識的裂痕進行手動標記，以便日後訓練 AI 模型來熟悉這些特徵。

容許我跳過技術名詞的解釋，因為說白了，這就是個有很多圖片，然後要試著把長得像菠蘿麵包的區域，用類似小畫家的工具圈選出來的任務。

只是你要很有毅力，因為圖片有很多張，幾乎翻不到盡頭，這些裂痕特徵，或者說菠蘿麵包，偶爾會出現在某些角落，你需要很有耐心的把它們一個個標記下來。認真的標註往往一個下午就過了，就這樣持續數個禮拜，是個蠻吃專注力的任務，尤其是像暑假這樣的熱天。

在這期間，我發現我看世界的方式變了。當我吃菠蘿麵包的時候，我會聯想到那些圖案。當我看到牆上的馬賽克藝術時，我會聯想到那些圖案。當我走在路上，我會聚焦在地上的石頭路，然後聯想到那些圖案。對了，還有便利店放在架上的方塊酥也是。總之，我突然變得很擅長辨識生活中這些與裂痕相似的紋路。

聽起來蠻好笑的，但我現在相信那些長時間專注在某些事情的人們，他們看世界的角度應該也很不一樣吧！
