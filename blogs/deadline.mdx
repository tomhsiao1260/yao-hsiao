# 最後期限

期末考、成果發表、年底目標，生活中有著大大小小的最後期限。在死線來臨前，事情的進展往往是快速且混亂的。想在這段期間快速推進，有幾個要點：

<br/>
<div> {`1. 在極短的時間內，明確的目標，能更清楚知道瓶頸的位置，在混亂和非常不適的嘗試中，讓情況漸漸變好，最後達成看起來不太能如期完成的事`}</div>

<br/>
<div> {`2. 過去幾個月漫長的嘗試，往往會在意料之外的環節派上用場，大大降低了離開混亂情況需要的時間`}</div>

<br/>
<div> {`3. 在共同目標的群體裡，別人的進展能幫助到你，你的進展也可能解決別人的瓶頸。這樣的回饋迴路能大幅提升群體接近目標的機會，也是團隊合作有趣的地方`}</div>
