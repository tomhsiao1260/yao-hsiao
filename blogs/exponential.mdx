# 聊聊指數

想像一下你有張紙，上方有個指數函數，但只顯示了 x=5 以前的曲線。於是你拿起筆，憑直覺把曲線補齊到了 x=10。然後你發現，旁邊還有另一張圖，描述著同個函數，但 y 軸被改成對數刻度，變成一條筆直的線段。你一樣藉著目測，把圖形補齊到了 x=10。為了比較兩張圖的數值是否彼此吻合，你做了些計算，然後你可能會感到意外，原來指數的增長比想像的劇烈。

如果把紅綠燈的倒數改成指數遞減，那綠燈的最後五秒大概會像這樣：7775, 1295, 215, 35, 5, 0 (紅燈)

想像一下，你有台 [Powers of Ten](https://m.youtube.com/watch?v=0fKBhvDjuy0) 腳踏車，它能將空間取對數，往前騎能夠到銀河系，往後騎能夠到細胞內。
