# 聊聊指數

我們有辦法更直觀的感受到指數的增長嗎？

如果把紅綠燈的倒數改成指數遞減，那綠燈的最後五秒大概會像這樣：7775, 1295, 215, 35, 5, 0 (紅燈)，你會在幾秒前發現紅燈快要到了？

想像一下你有張紙，上方有個指數函數，但只顯示了 x=5 以前的曲線。於是你拿起筆，憑著直覺把曲線延伸到 x=10 的位置。然後你發現，旁邊還有另一張圖，描述著同個函數，但 y 軸被改成對數的刻度，也就是一條筆直的線段。你一樣藉著目測，把線段延伸到 x=10。為了比較兩張圖的數值，你做了些計算，看看自己憑直覺的誤差有多少，然後你可能會感到意外，原來指數增長比想像中的劇烈。

想像一下，你有台 [Powers of Ten](https://m.youtube.com/watch?v=0fKBhvDjuy0) 腳踏車，它能將空間取對數，往前騎能夠到銀河系翱翔，往後騎能看到細胞尺度的風景。在晃了幾趟後，你拿起了手邊的對數地圖，標記著剛剛看到的地標和景色。

如果把世界的重要里程碑 [Canonical Milestones](https://upload.wikimedia.org/wikipedia/commons/2/2b/PPTCanonicalMilestones.jpg)，在時間上取對數，並濃縮成三天體驗，會是什麼感覺？第一天早晨，才剛看到銀河系的形成，下午就寒武紀大爆發，接著是夜晚的大滅絕，以及凌晨的猩猩崛起。第二天早上，你學著使用石器，保護火堆，並在睡前自創一些文字和壁畫。第三天，你開始種田、做輪子和發明數學，接著是文藝復興和工業革命。很快就到了夜晚，你逐漸回到了現代，近代物理和計算機的蓬勃發展。你躺在了床上，回顧著這幾天發生的事，想著明天早上要做些什麼？
