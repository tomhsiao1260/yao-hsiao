# 問題與學習

這篇我想以「問題」為主軸，看看它是怎麼在學習過程發揮作用的。

## 寫下問題

我蠻喜歡接觸新領域時的感受，它伴隨著許多問題的浮現，並且能激發好奇心，朝著自己感興趣的方向探索。然而，在很多學習環境裡，這些剛浮現的問題會快速消逝，因為在那些環境裡，學習更像是「模仿」：我們會很快的接受這個領域的思考框架，並在不久後問出跟專家們一樣的問題。到了這個階段，無法用特定方式思考或描述的問題，都會被視為奇怪的甚至是「蠢問題」。這其實蠻可惜的，因為當你成為一個領域的專家，跳脫框架去思考會變得困難。這時候教別人自己知道的知識，或是學習其他領域的知識會有些幫助。

但這仍然無法消去我對問題消逝的失落感：最初的學習感受？當時在想什麼？感到好奇的點？它可能是幾年前的第一堂課，剛接觸到的觀點，又或是意外的發現。雖然知道這對跳脫框架去思考有幫助，但當時想到了哪些「蠢問題」，也已經忘得差不多了。

這促使我開始把問題寫下來，並讓它在學習的過程中演化。這很有趣，因為這表示問題本身成為了主角，但它不需要急著被問出來，或是得馬上獲得解答。相反的，它能引導你去探索，為知識建立起關聯，琢磨出更好的問題，並逐漸養成對這個領域的獨到見解。
