# 拿起畫筆的時機

畫筆可以在紙張上繪製各式各樣的符號圖案，靈活的表達一些概念之間的關聯，甚至能增進我們進一步的理解。下面我想以幾個拿起畫筆的時機，來說明畫畫作為一種表達和理解工具的潛力。

## 實體素描

由於最近對骨骼的表面型態感興趣，促使我開始繪製骨骼素描，發現這個過程對物體型態的深度理解很有幫助。

那麼物體型態的深度理解是什麼意思呢？為了能更清楚的解釋，讓我花點篇幅來描述一下整個繪製的過程。雖然是第一次學習這類主題，但不難發現骨骼型態的複雜多變，以及關節接合處的許多細節。為了能有初步輪廓，我花了點時間查找了特定骨骼在不同角度的照片和對應名稱，以及這些局部是怎麼被組織起來的。這些前置作業大大的降低了下筆門檻，也不知不覺的學會了相關的基礎知識。接著在繪製骨骼輪廓時，我會把具指標性的特徵先抓出來，這讓我能更方便的補齊剩下的輪廓細節，也對整個骨架和局部形狀有更多的掌握。最後是上色，這讓我把注意力轉到了骨骼表面微妙的幾何轉變，並把深淺線條照自己的理解描繪了出來。

其實整個描繪過程都圍繞在不同層面的觀察上，這加深了我對骨骼型態的深度理解，甚至有許多細節是因為要動筆畫下才意外觀察到的。這種感覺有點像是，先透過地圖了解一座城市，接著慢慢限縮範圍到某條街道，然後實際走過裡頭的每個巷弄，所獲得的意外收穫。

## 抽象關係的視覺化

前個章節，我們看到了畫畫的過程是怎麼幫助到理解的，這個章節，我會著重在畫畫作為一種表達方式的環節，下面兩個例子是我想表達抽象關係的時機：

第一個例子比較偏藝術的表達，我嘗試視覺化被譽為最優美的數學定理 —— 歐拉恆等式。

<p align="center" style={{ height: '500px', padding: '10px' }}>
  <img src="./math.jpg" alt="math" style={{ width: 'auto' }} /> 
</p>

這就是著名的歐拉恆等式：

$$
e^{iπ} + 1 = 0
$$

歐拉將 5 個最基本的數學常數 (1, 0, π, i, e) 簡潔的聯繫起來，此外還用了 3 個重要的算術運算 (加法, 乘法, 指數運算)。這讓我很想以自己的方式來重新表達這些符號之間的巧妙聯繫所帶給我的震撼。

比起創作的過程，前期的構思是最困難的。在實際動筆前，我整整花了一個禮拜想像適合的構圖，以及怎麼把這些符號給圖形化。前幾天幾乎在為 -1 和 i 這兩個符號苦惱，一直無法找到適合的圖象表達。直到我發現 -1 能透過 0 到 1 的反向來表示，這讓我能 “畫“ 出一個面積為 -1 的正方形，並將它的邊長想像成虛數 i 的視覺化。雖然不是嚴謹的數學表達，但這很快讓我能構思出一幅完整的景象，並拿起畫筆，進入了創作的過程。
