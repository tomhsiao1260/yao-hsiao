# 拿起畫筆的時機

畫筆可以在紙張上繪製各式各樣的符號圖案，靈活的表達一些概念之間的關聯，甚至能增進我們對事物的理解。下面我想以幾個拿起畫筆的時機，來說明畫畫作為一種表達和理解工具的潛力。

## 實體素描

由於最近對骨骼的表面型態感興趣，促使我開始繪製骨骼素描，發現這個過程對物體型態的深度理解很有幫助。

那麼物體型態的深度理解是什麼意思呢？為了能更清楚的解釋，讓我花點篇幅來描述整個繪製的過程。雖然是第一次學習這類主題，但不難發現骨骼型態的複雜多變，以及關節接合處的許多細節。為了能有初步輪廓，我花了點時間查找了特定骨骼在不同角度的照片和對應名稱，以及這些局部是怎麼被組織起來的。這些前置作業降低了許多下筆門檻，也不知不覺讓我學會了相關的基礎知識。

接著進入繪製的階段，我會把具指標性的特徵先抓出來，這讓我能更方便的補齊剩下的輪廓細節，也對整個骨架和局部形狀有了更多的掌握。最後是上色，這讓我把注意力轉到了骨骼表面微妙的幾何轉變，並把深淺線條照自己的理解描繪出來。

也許你已經注意到了，整個描繪過程都圍繞在不同層面的觀察上：背景知識、骨架結構、輪廓細節...等，這加深了我對骨骼型態的深度理解，甚至有許多細節是動筆畫下才意外觀察到的。這種感覺有點像是，先透過地圖了解一座城市，慢慢限縮到某條街道，並走過裡頭的每個巷弄後，所獲得的意外發現。

## 抽象關係的視覺化

前個章節，我們看到了畫畫的過程是怎麼幫助到理解的，在這裡，我會著重在畫畫作為一種表達方式的環節。下面兩個例子是我想表達抽象關係的時機：

第一個例子比較偏藝術的表達，我嘗試視覺化被譽為最優美的數學定理 —— 歐拉恆等式。

<p align="center" style={{ padding: '10px 0 10px 0' }}>
  <img src="./math.jpg" alt="math" style={{ width: 'auto' }} /> 
</p>

歐拉將 5 個最基本的數學常數 (1, 0, π, i, e) 簡潔的聯繫起來，此外還用了 3 個重要的算術運算 (加法, 乘法, 指數運算)。這讓我很想以自己的方式來重新表達這些符號之間的巧妙聯繫所帶給我的震撼。

比起創作的過程，前期的構思是最困難的。在實際動筆前，我花了整整一個禮拜想像適合的構圖，以及怎麼把符號給圖形化。前幾天幾乎在為 -1 和 i 這兩個符號苦惱，一直無法找到適合的圖象表達。直到我發現 -1 能透過 0 到 1 的反向來表示，這讓我能 “畫“ 出一個面積為 -1 的正方形，並將它的邊長想像成虛數 i 的視覺化。雖然不是嚴謹的數學表達，但這很快讓我能構思出一幅完整的景象，並拿起畫筆，進入了創作的過程。

第二個例子，是我在參與一個叫 [Vesuvius Challenge](https://scrollprize.org/) 挑戰所做的演算法視覺化。這是個跨領域的考古科學挑戰，期望能透過群眾和現代科技的幫助，破譯古代卷軸內部的古希臘文字。過程中我們需要花很長的時間，在大量的電子掃描資料裡尋找可用的線索。這非常的耗時且不直觀，促使我開始思考有沒有更好的方式能初步的閱覽這些資料，進一步催生了基於場這類概念的視覺化雛形。

我不會在這裡解釋太多原理細節，不過核心概念是：透過自定義的電極產生的假想電場，讓使用者能順著場的方向閱覽這些資料。在這個想法誕生的當下其實蠻雀躍的，希望能分享給社群的其他人，以加速這個方向的進展。然而，我很快發現用文字表達這類概念的限制，這促使我拿起了畫筆，把腦海裡想像的概念具體的描繪出來。

## 其他拿起紙筆的時機

除了畫圖外，紙筆的操作提供了靈活的表達空間，我偶爾會用它在時間緊迫的時候做比較複雜的表達，有時會帶來意外的新理解。好比說，去年年底為了準備 Vesuvius Challenge 最後一個禮拜的衝刺，我跟朋友畫了張社群間的人物關係圖，列出了其他人在做的嘗試和目標。才意外的發現，群體間有些人的進展能幫助到我們尋找卷軸標題，而我們的進展也可能解決他們在卷軸展開遇到的瓶頸。這樣的回饋迴路是我在拿起紙筆前，沒有預想到的新理解。

## 更人性化的紙筆

以上就是紙筆參與表達和理解的過程，但說實在的，有時候整個流程在體感上並不是那麼的流暢。好比說，雖然實體素描的繪製能促進對骨骼型態的深度理解，但這樣精準描繪的要求，有時反而放慢了理解的速度；以及，雖然為了能拿起畫筆表達，促使了我對抽象符號的思考，但這個過程整整花上了一個禮拜；此外，雖然新演算法的靈感促使我拿起畫筆，但我還是在為不同視角的表達煩惱，最後還是決定分成多張視圖來表示。

有沒有可能打造一個更人性化的紙筆，讓表達和理解的過程更加的流暢呢？

想像一下，你手上有個神奇的魔法紙和筆，或是更人性化的平板裝置，它能從少量的操作推測你可能想完成的事。好比說，你也許不想每次面對相似型態的骨骼，都重畫一次精準素描，而是希望畫幾筆簡單輪廓就能自動生成熟悉的骨骼型態，如果是三維的互動影像更好，甚至能靈活比較不同接合方式的協同運動，建立更深度的理解。又或是當我對符號這類抽象表達停滯不前時，它能根據我的直覺提供可能的建議，讓我能展開並比較不同構圖的可能性。對了，如果這些圖像能照我想表達的那樣，直覺的動起來，那就再好不過了！
