# 寶藏的位置

像是藝術家林布蘭的世界名畫，哲學家菲洛得摩斯的生前手稿，又或是年代久遠的棘龍化石。人們願意注入大量資金，用上最前沿的科技，將這些物品進行高解析度的數位備份，進一步分析並理解內部可能隱含的訊息。我們普遍認為這些物品是很有價值的，因為如果能買賣的話，你不難從中換取到一筆可觀的資金。

像是小時候不經意在牆上的塗鴉，過去準備考試留下的手寫筆記，又或是散落在家中牆角的娃娃玩具。這些東西對其他人來說沒什麼價值，你也習慣將它們散落在家中的某個角落。但這些東西有時候卻會像回憶入口般，在某個悠閒的午後，偶然讓你想起了過去發生的趣事。好比說，啊我想起來了，這個塗鴉是小時候畫的，因為當時看到某個電視上的卡通角色，還記得 ...。這些物品就像一顆顆的小石頭，激起了水面一波波的漣漪，又像是通往過去記憶的連結。

但這些記憶連結是會消失的，想像一下你在家中不經意翻到一本阿伯的相簿，你不太知道這些照片是在哪裡拍的，當時的心情如何，還有一系列當時情境的細節等等。於是你打算打開相簿，直接跟阿伯一起看這個相簿，然後他也很樂意跟你分享那時候的故事，聽起來還蠻簡單的，對吧。但如果阿伯已經過世了，這些相簿上的照片就算用在高解析度的掃描進行數位備份，也無法找回過去這些照片背後的連結。
