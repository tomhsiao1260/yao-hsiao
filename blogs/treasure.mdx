# 寶藏的位置

像是藝術家林布蘭的世界名畫，哲學家菲洛得摩斯的生前手稿，又或是年代久遠的棘龍化石。人們願意注入大量資金，用上最前沿的科技，將這些物品進行高解析度的數位備份，進一步分析並理解內部可能隱含的訊息。我們普遍認為這些物品是很有價值的，因為如果能買賣的話，你不難從中換取到一筆可觀的資金。

像是小時候在牆上的塗鴉，過去考試留下的手寫筆記，又或是散落在家中的娃娃玩具。這些對其他人沒什麼價值，我們也習慣將它散落在家中角落。但這些東西卻能像回憶入口般，在某個悠閒的午後，讓你想起過去發生的一些事。好比說，啊，想起來了，這張塗鴉是小時候看卡通時，覺得很有趣所以畫的，這讓我想起那時候的一些事...等等。對我來說，這些物品就像通往過去的記憶連結，它們有如丟入水中的小石頭，激起了一波波名為回憶的漣漪。

但這些記憶連結不見得會一直存在。想像一下，你在倉庫翻到一本阿伯過去的相簿，於是你好奇的翻了一下，但除了照片上的影像，你很難知道更多資訊，像是照片的故事或當下的心情等等。如果阿伯已經去世了，也許你只能從照片和親戚間的印象，免強的拼湊出一些事情。整本相簿就像被封存般，通往那些故事的連結早已不復存在，再前沿的科技可能也愛莫能助。但相反的，如果阿伯還在的話，那這段記憶連結的訪問方式可能意外的簡單，通關密語大概會是：「嘿，阿伯，這張照片是在哪裡拍的啊？」
