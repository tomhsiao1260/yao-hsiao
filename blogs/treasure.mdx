# 寶藏的位置

像是藝術家林布蘭的世界名畫，哲學家菲洛得摩斯的生前手稿，又或是年代久遠的棘龍化石。人們願意注入大量資金，用上最前沿的科技，將這些物品進行高解析度的數位備份，進一步分析並理解內部可能隱含的訊息。我們普遍認為這些物品是很有價值的，因為如果能買賣的話，你不難從中換取到一筆可觀的資金。

像是小時候在牆上的塗鴉，過去考試留下的手寫筆記，又或是散落在家中的娃娃玩具。這些對其他人沒什麼價值，你也習慣將它們散落在家中角落。但有時候這些東西卻像回憶入口般，在某個悠閒的午後，偶然讓你想起了過去發生的趣事。好比說，啊我想起來了，這個塗鴉是小時候畫的，因為當時看到某個電視上的卡通，還記得 ...。這些物品就像一顆顆小石頭，激起了水面的漣漪，又像是通往過去記憶的連結。

但這些記憶連結是會消失的，想像一下，你在家中不經意翻到一本阿伯的相簿，但不知道這些照片有什麼特別意義，好比說拍照當時的心情或情境等等的細節。假設那位阿伯已經去世了，你只能從照片，和旁人的口述，嘗試拼湊一些片面事實。整本相簿就像被封存般，讓你難以訪問物品背後的情境故事，再前沿的科技可能也愛莫能助。但相反的，如果阿伯還在的話，那連結的訪問方式可能意外的簡單，通關密語大概會是：「嘿，阿伯，這張照片是在哪裡拍的啊？」
