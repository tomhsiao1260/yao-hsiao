Vesuvius Challenge 這個挑戰滿一年了，想藉這個機會寫下我當下的感受和學到了什麼。

<br />

首先，很開心看到 Vesuvius Challenge 官方把我的名字放到一張分工的圖表上，剛好藉著這個機會來回顧我對這個挑戰的想法。
對我來說，這項科學挑戰是許多人不斷嘗試和探索交織而成的，有些方法成功有些失敗，有時候一個方法的出現也可以讓另一個人得到啟發並在這個方向努力。
這樣的體驗常常在社群中彼此的討論中出現，以我的角度來看的話，最早是在去年三月知道這個挑戰的，那時候整個挑戰社群還很小，挑戰者間彼此都算認識。
印象最深刻的是 Luke 在很早就做出了一個搭配 Python 和 Javascript 的 Volume 視覺化雛形，我看到後得到了一些啟發，並開始打造 Segment Viewer。
雖然之後他就沒繼續做這塊，而是轉往機器學習辨識那塊去探索，但如果沒有他早期的嘗試，我也許也不會因為得到一些啟發而打造相關的工具。
另外，就是 Casey 天外飛來一筆的發現，催生了 Luke 開發相關的機器學習辨識工具，這也主導了往後好幾個月大家的研究方向。
這段對我來說印象最深刻的有幾點，一個就是 Casey 憑藉自己的強大視覺在龐大的資料海裡找到 crackles 的線索，
記得當時的主流方向是沿用 Kaggle 上表現不錯的算法，但一直難有所突破，而當時 Casey 則是選擇長時間守在電腦前，憑藉肉眼嘗試找出那些不一定會有成果的蛛絲馬跡，真的是很驚人的毅力。
而當 Casey 發表他的發現時，當時整個社群的討論還不是很明朗，Luke 是第一個非常重視 Casey 結果的人，並在他的發現上嘗試建造相關的算法，這段我也對 Luke 的直覺印象深刻。
那時候我就在想，這段肯定是官方原本意料之外的發展，因為這表示我們可能需要捨棄 Kaggle 上大家的成果來另外打造一套模型，結果 Youssef 就在之後幾個月長時間比較不同模型的優缺點，並成功把大家在 Kaggle 上的努力銜接上了 Casey 他們的貢獻。
而我也在這段時間做了不少跟視覺化有關的嘗試，這些工具也讓 Segment 團隊之間更了解哪些區域尚未裁切，而他們裁切的進展也幫助到機器學習辨識能力的改進。
然後 Djosey 進一步把這些文字拼接在一起也給了我一些啟發，促使我跟另一個高中生 Fish 一起做 VC Whiteboard 的開發，我也在合作的過程中學到了很多。
這也讓我體會到，這個挑戰之所以成功，Grand Prize 得獎者的誕生，都是一系列這樣的插曲和嘗試集體交織而成的，如果中間少了中間幾個環節，或社群間少了些不起眼的嘗試或失敗，之後的發展可能都會很不一樣。

另一個讓我印象深刻的是，官方或者說前 Github 創辦人 Nat Friedman，在發起這個挑戰上時程規劃的縝密。
事後看完了他的規劃，再回想自己去年參加挑戰的過程，有種以另一個視野重新看同一件事的感覺。
好比說官方為了加速社群間的研究進展，他們會在短期上，每幾個月有個階段性的開源獎項，這能定期更新參賽者間彼此的研究進展，也能避免挑戰者間為了更長遠 Grand Prize 的競爭而延後發表自己的成果。
此外，Nat 他們每週也都會問自己一個問題：為了最大程度地增加某人贏得得最大獎 Grand Prizes 的機會，現在可以做的最好的事情是什麼？
另外，他們也給 Segment 團隊一個重要目標是要多跟參賽者密切交流或給予幫助。
聽到這件事的第一時間我的想法是，如果能更早了解到這些思路，就能更清楚其他區塊的人想要什麼，我在什麼位置，以及他們的目標，我就更有動力聚焦在真正重要的方向上。
好比說如果我知道 Nat, JP 每個禮拜都在想如何提高年底獲得 Grand Prizes 的某個人的機率，那我可能會全心去幫助這些人獲得這些獎項，比較不會失焦。
以及如果我知道 Segment 團隊的目標是跟我們密切的合作，那我可能就會更勇於問他們問題，而不是怕他們覺得麻煩而錯失寶貴的資訊，也會更主動把他們的東西用來打造工具等等。
此外，我也對官方拆解問題的方式印象深刻，除了具體卻不過於制式的需求清單，讓我能很大程度在自己感興趣的方向上自由探索以外，Nat 也很清楚知道哪個階段要做些什麼。
好比說，他如果要直接挖開整棟高級別墅進行重建工作，或者說尋寶，聽起來很有趣但成本和不確定性過高，也很難拿到外部資金的注入。但如果是先找別墅裡的卷軸，修復尚未出土的古代文獻，反而能把各種領域有趣的人才聚集在一起，資金也隨著社群的進展逐漸注入。
而且到目前爲止，單一卷軸的辨識還很花時間，Nat 剛好可以利用這段時間來規劃醞釀之後更龐大的挖掘和掃描工作，進而慢慢累積到足夠的資金把整個豪華別墅重建出來。
我也能感覺到逐步拆解一個遠大願景並一步步實踐它的威力。
還有另一點是，每個階段的里程碑都很重要，因為這些都是下階段進展的基礎，透過這個過程挑戰者間也可以持續互相學習，Nat 也對這一系列複雜問題有個很有趣的切入點，整個過程讓我印象深刻。

<br />

最後，讓我最驚喜的就是 Vesuvius Challenge 官方的 JP 在最近來到台灣跟我們見面這件事。
仔細想想，如果我去年沒有持續參加這個挑戰，JP 就不會來台灣，這塊島嶼應該就不會跟 Vesuvius Challenge 有所交集。
我似乎也在這個世界的洪流中改變了點什麼，也透過這個挑戰認識了來自世界各地有趣的人。
一個非常困難的問題也許會讓單獨一個人望之卻步，但這些問題卻很自然的把一群喜歡挑戰的人聚集再一起，互相學習並嘗試解決這些問題，這就是我參加 Vesuvius Challenge 滿一年後的感覺！
